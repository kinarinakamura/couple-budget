# Couple Budget

カップル向け家計簿アプリケーション

## 技術スタック

- **Backend**: Laravel 12 (PHP 8.2+)
- **Frontend**: Vue 3 + Inertia.js
- **CSS**: Tailwind CSS + DaisyUI
- **Chart**: Chart.js + vue-chartjs
- **Database**: SQLite
- **Build**: Vite

## 開発コマンド

```bash
# 開発サーバー起動
composer dev
# または個別に
php artisan serve   # http://localhost:8000
npm run dev         # Vite

# テスト
composer test

# ビルド
npm run build
```

## ディレクトリ構成

```
app/           # Laravel アプリケーション
resources/js/  # Vue コンポーネント・ページ
routes/        # ルーティング
database/      # マイグレーション・シーダー
```
